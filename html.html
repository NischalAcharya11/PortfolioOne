HTML — Semantic Structure

HTML defines the meaning and structure of content using semantic tags. Semantic HTML improves accessibility (screen readers), SEO, and maintainability.

Semantic article example
<article class="post">
  <header>
    <h2>My First Post</h2>
    <p><time datetime="2025-08-24">Aug 24, 2025</time></p>
  </header>
  <section>
    <p>Accessible and semantic structure helps everyone.</p>
  </section>
  <footer>
    <button>Like</button> <button>Share</button>
  </footer>
</article>

Key Semantic Elements

  • <header>, <main>, <footer>
  • <article>, <section>, <aside>
  • <nav> for navigation landmarks
  • Accessible attributes like aria-*, alt
